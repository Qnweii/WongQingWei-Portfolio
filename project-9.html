Project:

Bermuda Triangle Prototype Design

Project Type
Individual project
Project Goals
To design the prototype of the cardboard game assets
for a new game to be launched in 2022.
Chosen theme is 'Bermuda Triangle'
Project Task
To design 4 pieces of Cardboard Game Card, a Cardboard Game Board and a poster with the selected theme.
Tool
Adobe Illustrator
Duration
2 weeks

OVERVIEW

Background

Designing a prototype cardboard game with the theme "Bermuda Triangle" was a challenge for me
as I had to develop and create the most appropriate illustration to fit the theme.
Because this theme was so vague and mysterious, I looked at a variety of concept art online to get the idea as close as possible, so that the overall look is consistent and easy to understand.
While keeping in mind to incorporate gamification elements, such as adding/subtracting points, card usage, and description.
I also used the same default color and tone throughout in order for the prototype to fit in and resemble an entire collection of theÂ game prototype. Overall, the 'Bermuda Triangle' theme was intended to express mystery, hence, this prototype design was designed to be simple yet mysterious.
