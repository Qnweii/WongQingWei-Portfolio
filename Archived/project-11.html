Project:

Iron Giant Character Modelling and Rigging

Project Type
Individual project
Project Goals
To familiarization in the creation process of functional 3D assets
suitable for real time applications and rigging process.
Project Task
To create and build up a robot character model through organic modeling and rigging approach in Autodesk Maya.
Tool
Maya
Substance Painter
Duration
3 weeks

OVERVIEW

Background

This was my first project modeling and rigging a character in Maya. This project served as the beginning of my modeling process, during which I explored a variety of approaches and gained new techniques and skills from the rigging process as well, such as understanding IK Rigging in Maya.

This character's modeling and rigging served as the foundation for my learning in Maya and Substance Painter, on how to apply decals for texturing.

When modeling this Iron Giant, I also tried to keep the poly count as low as possible while paying attention to the details of its structure to make the model as accurate as possible from the actual Iron Giant robot.
