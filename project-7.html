Project:

'Sushi Rush' Game Controller

Project Type
Individual project
Project Goals
To design a physical set of controllers
that will act as an input device for an online browser-based game.
The main consideration is on how wholesome an experience
can provide for the users.
Project Task
To conceptualise, propose and construct a set of customised physical controllers meant for exploring an online browser-based game.
Tool
Air Dry Clay
Adobe Illustrator (poster designs)
Duration
2 weeks

OVERVIEW

Background

To use a set of Makey Makey Classic, coupled with own materials to construct the physical controllers. The look and feel of the controllers should match the concept / theme for the chosen game, Sushi Rush.
